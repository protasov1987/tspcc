# Target Architecture

Этот файл оставлен как compatibility entry point для старых ссылок.

Миграция из `docs/architecture/migration-plan.md` завершена, поэтому бывшая
target architecture теперь является актуальной архитектурой проекта и
зафиксирована в:

- [Current Architecture](./current-architecture.md)

Для новых изменений, ревью и архитектурных проверок обязательным источником
истины является `docs/architecture/current-architecture.md`.

`target-architecture.md` больше не является отдельным планом будущего перехода
и не должен использоваться как основание для временного расширения legacy-кода.
